# 🌱 GreenSteps

> Small Steps. Lasting Impact.

GreenSteps is a sustainability habit-building platform that helps users take simple daily actions to create a greener future. Instead of overwhelming users with carbon metrics and technical jargon, GreenSteps focuses on behavior change through habit tracking, forest growth, gamification, AI-powered guidance, and family participation.

---

## 📖 Overview

Most sustainability platforms focus on reporting environmental impact.

GreenSteps focuses on helping people build better habits.

Users complete eco-friendly actions, earn rewards, grow their virtual forest, and receive personalized recommendations from GreenBuddy AI.

The goal is to make sustainability accessible, enjoyable, and rewarding for everyone.

---

## ✨ Key Features

### 🌳 Forest Growth System

Start your journey as a seed and grow your virtual forest through sustainable actions.

Progression:

Seed → Sprout → Plant → Young Tree → Tree → Large Tree → Forest

---

### 📊 Activity Tracking

Track daily activities across multiple categories:

* 🚗 Transportation
* 🍔 Food
* ⚡ Energy
* 💧 Water
* 🛒 Shopping
* ♻️ Waste

Designed for quick logging in under 10 seconds.

---

### 🎯 Daily Challenges

Complete sustainability challenges such as:

* Walk 15 minutes
* Use a reusable bottle
* Turn off unused lights
* Reduce water usage
* Plant a tree

Earn XP, badges, and rewards.

---

### 🤖 GreenBuddy AI

Your personal sustainability assistant.

GreenBuddy helps users:

* Build eco-friendly habits
* Save energy
* Reduce waste
* Set sustainability goals
* Track progress

---

### 👨‍👩‍👧‍👦 Family Mode

Create shared family forests.

Work together to:

* Complete challenges
* Earn rewards
* Reach milestones
* Build sustainable habits

---

### 🏆 Gamification

* XP System
* Achievement Badges
* Daily Streaks
* Forest Levels
* Community Challenges

Many people want to help the environment but:

GreenSteps transforms sustainability into a simple daily habit through motivation, rewards, and visual progress.

## 🚀 Getting Started

### Prerequisites

Node.js installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yuval0408/greensteps.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
